### App Store Connect 設定手順

1. [App Store Connect](https://appstoreconnect.apple.com) にログイン
2. 「マイ App」→「献立カート」→「App Store」タブを開く
3. 「日本語（日本）」の言語セクションを開く
4. 下記の各フィールドに値をコピペして入力する
5. 「保存」→「申請に追加」

### コピペ用テキスト

#### アプリ名（変更不要）

```
献立カート
```

#### サブタイトル（30文字以内）— 残り14文字の余裕あり

```
週間献立と買い物リストを自動作成
```

文字数: **16文字**（制限: 30文字）

#### プロモーションテキスト（170文字以内）— 審査なしでいつでも変更可能

```
献立を先に決めてから買い物へ。食費の無駄と食材ロスを減らすシンプルなアプリです。完全無料・オフライン対応。
```

文字数: **41文字**

#### キーワードフィールド（100文字以内、カンマ区切り・スペースなし）

```
献立,買い物リスト,食費節約,週間献立,料理計画,食材管理,献立表,節約,食費,料理
```

文字数: **46文字**（制限: 100文字）— 残り54文字に追加可能なキーワード候補: `夕飯,昼食,家計,食材,レシピ,まとめ買い,食費管理`

**選定理由:**

- 「献立」「買い物リスト」— 競合が少なく上位表示しやすい
- 「食費節約」「節約」— 検索ボリュームが大きい
- 「料理計画」「食材管理」— ニッチで転換率が高い
- アプリ名・サブタイトルに含まれる単語は除外（重複カウントされない）

#### 説明文（4,000文字以内）— 冒頭3行が最重要

```
「今日の夕飯どうしよう」を解決する、シンプルな献立＆買い物アプリ。

週の献立を決めるだけで、買い物リストが自動で完成。
余計な食材を買わず、食費と食材ロスを減らせます。

【主な機能】
・週間献立カレンダー（月〜日）
・食材リスト自動集計（重複まとめ）
・過去の料理履歴から検索・再利用
・完全オフライン対応（通信不要）
・データのバックアップ＆復元

【こんな方におすすめ】
・食費を減らしたい方
・食材を余らせてしまう方
・毎日「今日何作ろう」と悩む方
・シンプルなアプリが好きな方

広告非表示は ¥300 の買い切りで対応しています。
```

## 5. アプリ内グロース施策

### レビュー誘導ダイアログ

`SKStoreReviewAPI` を使い、以下のタイミングで表示する：

| トリガー | 条件 |
|---|---|
| 継続使用 | アプリ起動 7日後（かつ買い物リスト再生成が 3回以上） |
| IAP 購入後 | サンクス画面の「閉じる」後にワンアクション表示 |

> **注意:** Apple の制限により同一ユーザーには年 3回まで。乱用しない。

実装場所: [src/hooks/useAdManager.ts](src/hooks/useAdManager.ts) に追記予定（現在は広告管理のみ）

### IAP 購入後フロー

購入完了後に「ありがとうございます。よろしければ App Store にレビューをお願いします」のモーダルを表示。レビューボタンタップで `SKStoreReviewAPI` を呼ぶ。
